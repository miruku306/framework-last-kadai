# 📀 レンタルビデオ在庫管理システム

DjangoとPostgreSQLを使用して開発した、レンタルビデオ店向けの在庫管理Webアプリケーションです。

専門学校のフレームワーク開発授業の最終課題として制作しました。

---

# 📖 概要

レンタルビデオ店で扱う作品の在庫状況を管理することを目的としたWebアプリケーションです。

作品の登録・編集・削除・一覧表示ができ、現在の在庫状況をダッシュボードで確認できます。

また、ジャケット画像の登録にも対応しています。

---

# ✨ 主な機能

- ビデオ一覧表示
- ビデオ新規登録
- ビデオ編集
- ビデオ削除
- 商品コード自動採番
- ジャケット画像アップロード
- 在庫状況表示
    - 在庫あり
    - 残りわずか
    - 在庫切れ
- ダッシュボード表示
    - 総作品数
    - 貸出中本数
    - 在庫切れ作品数
- Bootstrapによるレスポンシブデザイン

# 📸 画面イメージ

# 🚀 起動方法

## Docker起動

```bash
docker compose up -d
```

## Django起動

```bash
python manage.py runserver 0.0.0.0:8000
```

ブラウザでアクセス

```
http://127.0.0.1:8000/
```
