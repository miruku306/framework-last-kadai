### 第１回の構築ができなかった人
- 第１回のDockerの準備をすすめる。（フォローするので声かけて）<br>
ワークスページに構成を追加するがでてコンテナが作られない場合は、.devcontainerと構成ファイルが見つけられない場合が多いです。「ファイル」→「フォルダを開く」で開いたフォルダのトップに.devcontainer/devcontainer.jsonがある必要があります。<br>
どうしてもダメな場合は新たにフォルダを作って、VSCodeで開きなおしそこで作業してみてください。<br>
また、devcontainer.jsonで黄色い波線（警告）が出ることがありますが、コンテナ内で拡張機能がインストールされると消えると思います。
